# General Intelligence Systems — Claude Code Plugins

## Install

Add the marketplace in Claude Code:

```
/plugin marketplace add general-intelligence-systems/plugins
```

Then install a plugin:

```
/plugin install bpm@general-intelligence-systems
```

Or browse everything with `/plugin`.

## Available plugins

| Plugin | Description |
| --- | --- |
| `bpm` | BPM tooling for Claude Code: author bpmn-io form-js form schemas (create/edit forms, validation) that form-js-viewer can render. |
| `ruby-development` | General Ruby development tooling for Claude Code: skills for idiomatic modern Ruby, starting with HTTP clients via the http.rb gem (chainable API, response pattern matching, persistent connections). |
| `ruby-gtk` | Ruby GTK4/Libadwaita tooling for Claude Code: build, review, and plan Ruby GTK4 desktop apps using the declarative memoized-widget pattern. |
| `rubycam` | Webcam control for Claude Code via the rubycam CLI: drive any UVC camera (pan/tilt/zoom, image controls, snapshots) plus OBSBOT Tiny 2 vendor features (wake/sleep, AI tracking, presets, HDR, exposure). |
| `cnstudio` | CnStudio design tools for Claude Code: MCP server connection, design-model skill, and slash commands. |
