---
description: Open a designed component's live page in the browser via CnStudio
argument-hint: "[component]"
---

Use the cnstudio MCP tools. Call `openInBrowser` with `$ARGUMENTS` as the
`component` when given (otherwise the current component opens). Report the
returned URL, and mention the canvas URL variant
(`<dev-server>/__cnstudio/?component=<name>`) for the editable view.
